# Змейка с препятствиями

Игра на HTML5 Canvas с сохранением рекорда в LocalStorage.

## Локальный запуск

```powershell
python -m http.server 8080
```

Откройте в браузере: http://localhost:8080

> Стили встроены в `index.html`, скрипт — `js/app.js` (без ES-модулей).

## Публикация на GitHub Pages

## Важно: как открывать сайт

| Способ | Работает? |
|-------|----------|
| `https://ЛОГИН.github.io/РЕПО/` (GitHub Pages) | ✅ |
| Просмотр файла `github.com/.../blob/.../index.html` | ❌ |

## Что загрузить в репозиторий

```
├── index.html      ← стили встроены внутрь файла
├── js/
│   └── app.js      ← один файл, без ES-модулей
├── .nojekyll
└── README.md
```

Папки `css/`, `main.js`, `game.js` больше не нужны для GitHub — достаточно `index.html` и `js/app.js`.

## Шаги

1. **Settings → Pages → Deploy from branch `main` → `/ (root)` → Save**
2. Подождите 1–2 минуты
3. Откройте `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

## Если игра не запускается

- Загружен ли файл `js/app.js`?
- Открываете ли вы именно GitHub Pages URL, а не просмотр файла в репозитории?
- На странице появится красное предупреждение, если вы открыли файл через GitHub blob-просмотр
